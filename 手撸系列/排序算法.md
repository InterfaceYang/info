# 							排序算法

##### 冒泡算法

```js
function boolSort(arr) {
  var len = arr.length;
  for(var j = 0 ; j < len -1; j++) {
    for(var i = 0 ; i < len-1; i++) {
      if(arr[i] > arr[i+1]) {
        [arr[i],arr[i+1]] = [arr[i+1], arr[i]]
      }
    }	
  }

  return arr
}
```

##### 快排

```js
function quickSort(arr) {
  if(arr.length < 1) {
    return arr
  }
  var left = []
  var right = []
  var flag = arr.shift()
  for(var i = 0 ;i < arr.length ; i++) {
    if(arr[i]  < flag) {
      left.push(arr[i])
    } else  {
      right.push(arr[i])
		}
  }
  return quickSort(left).concat(flag,quickSort(right))
}
```
